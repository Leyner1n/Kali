**Ctrl+Shift+T** - New Terminal

**Ctrl+L** - Clear

**Tab** - Auto Complete

**Shift+PgUp(Dn)** - Scroll Up(Dn)

**Tab** - Chose any variants

**Ctrl+C** - Kill the process

**Ctrl+Z** - Stop the process

**fg [number]** - foreground job (возобновить процесс)

**Ctrl+D** - Close Terminal


## Cursor ShortCuts

**Ctrl+r** -> Search command history

**Ctrl+a** -> Move cursor to beginning

**Ctrl+e** -> Move cursor to end

**Ctrl+k** -> kill = cut (from where the cursor is)

**Ctrl+y** -> Paste = Yank
