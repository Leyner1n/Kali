## **Vim - Strong text editor which supports on (Linux, MacOS, Windows)**

### Mods 
	1. Normal  (moving/copy/paste/delete)
	2. Insert  (writing)
	3. Visual (select text + m/c/p/d)
	4. Command-line (save/exit)
1. **Normal Mode**
		"h" -> Left
		"j" ->  Down
		"k" -> Up
		"l" -> Right
		"0" -> Begin of the line
		"$" -> End of the line
		"G" -> end of the file
		"gg" -> begin of the file
		"w" -> trough of the word straight
		"b" -> trough of the word back
